# Output Specification

Use this checklist to validate each generated practice package.

## Required Files

- `SKILL.md`: coaching-ready skill for the specific practice topic.
- `research.md`: query log, Tavily answers, and source inventory.
- `sources.json`: structured source metadata for auditing and reuse.

## `SKILL.md` Required Sections

- `Session Setup`
- `Research-Backed Guidance`
- `Practice Flow`
- `Coaching Rubric (1-5)`
- `Common Failure Modes`
- `Sources`

## Quality Gates

- Include at least 8 unique sources when possible.
- Prefer source diversity across multiple domains.
- Ensure guidance is specific to the topic, not generic coaching advice.
- Keep drills executable in a live conversation (clear prompt + success criteria).
- Keep rubric criteria observable and scoreable in-session.
